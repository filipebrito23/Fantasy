# NBA Google Sheets Automation

Arquivos incluídos:
- `Fantasy.py`: script principal para preencher todas as abas da planilha Google Sheets.
- `requirements.txt`: dependências Python.

## Como usar
1. Baixe sua credencial de service account do Google Cloud e salve como `service_account.json` no mesmo diretório do script.
2. Compartilhe a planilha com o e-mail da service account como Editor.
3. Instale dependências:
   `pip install -r requirements.txt`
4. Execute:
   `Fantasy.py` ou `app.py`, se quiser ver rodando para todas as abas
